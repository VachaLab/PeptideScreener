Currently shows how well peptides can be classified with different feature sets.

Feature generation options

ESM2, PBERT
Protein language model embeddings
PCHEM
Classic physicochemical descriptors, output will include feature importance
CUSTOM_FEATURES
Use your own pre-computed features from CSV, output will include feature importance. (Features have to be included in uploaded train an validation CSV files !)

Current constraints / issues

  • Labels must be binary: 0 / 1
  • Only Random Forest classifier is available for now
  • PLM approaches Overfit on training dataset
